Historia de Python

Diciembre 1989

Orígenes en el CWI

Guido van Rossum inicia Python durante las vacaciones de Navidad como sucesor de ABC. Quería un lenguaje potente, extensible y con acceso al sistema operativo.

Febrero 1991

Python 0.9.0 — El primer lanzamiento

Ya incluía herencia múltiple, funciones lambda, map/filter/reduce, manejo de excepciones, tipado dinámico y módulos. Una base sólida desde el día uno.

1994 — 2010

Era Python 1.x y 2.x

Python 1.0 trajo Tkinter. Python 2.0 añadió recolector de basura y Unicode. Python 2.7 (2010) fue la última versión de la rama 2.x, con soporte extendido hasta enero de 2020.

Diciembre 2008

La Gran Ruptura — Python 3.0

Versión incompatible con la rama anterior. Print pasó a ser función, la división de enteros retorna float, Unicode por defecto. La transición duró de 2008 a 2018.

2015 — 2020

Era Moderna temprana

3.5 trajo async/await. 3.6 introdujo f-strings. 3.7 añadió Dataclasses. 3.8 sumó el operador walrus :=. Cada versión pulió la expresividad del lenguaje.

2021 — 2023

Potencia y Velocidad

3.10 incorporó pattern matching (match/case). 3.11 logró hasta 60 % de mejora en rendimiento. 3.12 mejoró asyncio y los subinterpreters.

2024

Python 3.13 — El fin del GIL

Eliminación gradual (experimental) del Global Interpreter Lock, abriendo camino a verdadero paralelismo. Un hito histórico en la evolución del lenguaje.

Curiosidad

¿Por qué "Python"?

El nombre no proviene de la serpiente. Guido van Rossum era fan de Monty Python, el legendario grupo de comedia británico. Quería un nombre corto, único y misterioso. Así nació uno de los nombres más reconocibles en la historia del software.

Parte 02

Recursos del Ecosistema

Doce librerías esenciales que demuestran la versatilidad de Python, desde procesamiento de imágenes hasta inteligencia artificial y redes.

Pillow

pip install Pillow

Fork moderno de PIL. Redimensiona, aplica filtros, agrega marcas de agua, convierte formatos y preprocesa imágenes para proyectos de IA.

Imágenes

Scrapy

pip install Scrapy

Framework profesional para web scraping a gran escala. Maneja concurrencia, seguimiento de enlaces y exportación a JSON, CSV y bases de datos.

Web Scraping

NumPy

pip install numpy

Computación numérica de alto rendimiento con arrays multidimensionales. Base de Pandas, Scikit-learn, TensorFlow y gran parte del ecosistema científico.

Ciencia de Datos

Pandas

pip install pandas

Análisis y manipulación de datos tabulares con DataFrames. Lectura, limpieza, agrupación, fusión y análisis como Excel, pero a escala masiva.

Análisis de Datos

Scapy

pip install scapy

Manipulación de paquetes de red. Permite crear, enviar, capturar y analizar paquetes (ping, escaneo de puertos, ARP spoofing, fuzzing).

Redes / Seguridad

Matplotlib

pip install matplotlib

Visualización de datos: líneas, barras, histogramas, gráficos 3D y más. Es la base de Seaborn y otras librerías de visualización avanzada.

Visualización

Kivy

pip install kivy

Framework para apps móviles y multiplataforma (Android, iOS, Windows, Linux). Soporta multitáctil y renderizado con OpenGL.

Apps Móviles

NLTK

pip install nltk

Procesamiento de Lenguaje Natural: tokenización, stemming, lematización, análisis de sentimientos y etiquetado gramatical.

NLP

Keras

pip install keras

API de alto nivel para Deep Learning sobre TensorFlow. Fácil de usar para redes neuronales, visión por computadora, NLP y series temporales.

Deep Learning

SQLAlchemy

pip install sqlalchemy

ORM potente para bases de datos. Permite trabajar con objetos Python en lugar de SQL directo y soporta múltiples motores de BD.

Bases de Datos

Django

pip install django

Framework web "baterías incluidas". ORM, admin automático, autenticación, templates y seguridad. Usado por Instagram, Pinterest y más.

Web Framework

Twisted

pip install twisted

Framework para programación asíncrona y en red. Ideal para servidores de alta concurrencia como chats, juegos y streaming con un solo hilo.

Async / Redes

Python ha evolucionado desde un proyecto personal de Guido van Rossum en 1989 hasta convertirse en el lenguaje más versátil y popular de la actualidad. Su sintaxis clara, su filosofía, su comunidad y su enorme ecosistema de librerías lo hacen ideal tanto para principiantes como para proyectos complejos en IA, web, ciencia de datos, ciberseguridad y más.
